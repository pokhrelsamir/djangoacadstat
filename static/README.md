# Static Files

Project-wide static assets served by Django in development and collected for production.

## Contents

| Folder   | Purpose                    |
|----------|----------------------------|
| `css/`   | Stylesheets                |
| `js/`    | JavaScript files           |
| `images/`| Images and icons           |

Configured in `academicsys/settings.py` via `STATICFILES_DIRS`.
