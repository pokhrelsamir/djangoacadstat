1.Python virtual environment creation:
environment\setup_venv.bat
(or manually: python -m venv environment\venv)


Virtual environment activation:
environment\venv\Scripts\activate

To deactivate:
deactivate


2. Install Django

pip install Django



3.To update, 

run: python.exe -m pip install --upgrade pip


4. Integrate Django with PostgreSQL (Driver Installation)

pip install psycopg[binary]


To install all the pip from the list we can run

pip install -r environment\requirements.txt

that is being mentioned or written in environment\requirements.txt



5.Username and Password in PostgreSQLAdmin


CREATE USER username WITH PASSWORD 'password';
ALTER ROLE username SET client_encoding TO 'utf8';
ALTER ROLE username SET default_transaction_isolation TO 'read committed';
ALTER ROLE username SET timezone TO 'UTC';


CREATE DATABASE acadstatmain OWNER username;



6. Create superuser in Django

First migration helps table to be update in Django project

python manage.py migrate

If any issue in migration then you can prefer 

Option 1 : Making fake migrations

python manage.py migrate --fake


Option 2: Deleting all files from core/migrations except __init__.py

python manage.py makemigrations
python manage.py migrate


Force push your commit without caring about others latest commit as it overwrites through the file in GitHub

git push origin main --force


Then creation of superuser

python manage.py createsuperuser


7.QR code driver installation in project with Django

python -m pip install qrcode[pil] Pillow


8.How to Create Student Login Accounts:

cd djangoacadstat
python manage.py create_all_student_users --password mypassword



Student User Accounts for Login

Created two management commands:

Single student:

python manage.py create_student_user <student_id> <password>


All students at once:

python manage.py create_all_student_users --password student123



List of students in the database:

python manage.py list_students




AI Analysis Integration:

Option 1:

python -m pip install -r environment\requirements.txt
python manage.py migrate
python manage.py send_notifications --dry-run
python manage.py send_notifications --threshold 60



Option 2: Or if python doesn't work, trying as python3:

py -m pip install -r environment\requirements.txt
py manage.py migrate
py manage.py send_notifications --dry-run



Option 3:

Install requests
py -m pip install requests

Create migration for new email/phone fields
py manage.py makemigrations

Apply migration
py manage.py migrate

Try again
py manage.py send_notifications --dry-run



Teachers login ID and password setup

python manage.py create_teacher_user 1          # password defaults to "pass"
python manage.py create_teacher_user 1 mypass   # custom password


sync_teacher_users command (new) — fix all existing teachers at once:
python manage.py sync_teacher_users

For any new teacher added later
python manage.py list_teacher
python manage.py create_teacher_user <teacher_id>


9. Run python file
